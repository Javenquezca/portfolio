# Javier Vásquez — Portafolio

Portafolio personal con estética cybersecurity / terminal.

## Stack
- Next.js 16 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Desplegado en Vercel

## Secciones
- Hero con efecto glitch y partículas
- Sobre mí con typewriter
- Terminal interactiva con comandos
- Stack tecnológico con barras animadas
- Proyectos en carrusel
- Auditorías de seguridad
- Easter egg (Konami Code)

## Desarrollo local
npm install
npm run dev
